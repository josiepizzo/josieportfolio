Full Stack Web Developer

Josie Pizzolato, specializing in the development of HTML5, CSS3 and Responsive websites using valid and semantic coding practices. Highly motivated creative developer.

I've completed training to become a full-stack web developer through Rutgers University Coding Bootcamp, a rigorous yet exciting program where I learned in-depth web development including AngularJS, APIs, NodeJS, MySQL, Express, and so much more.

For the past 5 years I have been able to successfully navigate a path into front end developing working for some Fortune 500 companies, Dow Jones, Bristol-Myers Squibb, Vonage and most recent Prudential. As a contractor I have experienced working in a team environment and have been able to contribute to some amazing projects with the help of some really brilliant mentors. I learned so much on this path that it prepared me to making a commitment to pursue and successfully complete my Full Stack Certification.
